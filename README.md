# FieldSensing API

Go言語のフレームワークである[Gin](https://github.com/gin-gonic/gin)を用いてAPIを作成します

## Description(詳細)

Pluslab/fieldsensing-api -　圃場管理システムのAPI用のリポジトリ  
iOSやAndroidなどマルチプラットフォームに対応するために作成

## Requirements(開発環境)

|言語/FW|Version|
|:--|--:|
|go|1.10.3|
|docker|18.03.1|
|docker-compose|1.21.1|

## Usage(使い方)
`docker-compose up`

You can see the current program on the **localhost:8080**
